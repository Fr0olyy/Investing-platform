# Кратко о проекте: backend + ML

## Что это

Это учебная инвестиционная платформа с backend на FastAPI и встроенным ML-модулем для сценарного анализа цены акций.

Пользователь может:

- зарегистрироваться;
- получить виртуальный брокерский счет;
- смотреть реальные котировки;
- покупать и продавать активы;
- смотреть состояние портфеля;
- получать прогноз и сценарную цену актива.

## Из чего состоит проект

### Backend

Backend отвечает за:

- авторизацию и JWT;
- хранение пользователей и портфелей;
- операции покупки и продажи;
- работу с активами, котировками и свечами;
- расчет прибыли/убытка;
- выдачу API для frontend.

Технологии backend:

- Python 3.13
- FastAPI
- SQLAlchemy
- Pydantic
- PostgreSQL / SQLite
- Docker

### ML

ML-модуль отвечает за:

- сбор исторических данных;
- подготовку датасета;
- обучение модели по каждому тикеру;
- расчет прогнозной цены;
- сценарный анализ по макрофакторам.

Технологии ML:

- pandas
- scikit-learn
- joblib

## Какие данные используются

Реальные рыночные данные:

- акции и индексы: MOEX ISS
- USD/RUB и ключевая ставка: Банк России
- Brent: FRED

## Как работает ML

Модель использует признаки:

- предыдущая цена закрытия актива;
- Brent;
- USD/RUB;
- IMOEX;
- Key Rate;
- RGBI.

Алгоритм:

- линейная регрессия.

Результат:

- predicted price;
- impact percent;
- confidence score;
- drivers analysis.

## Что уже реализовано

- реальные котировки и исторические свечи;
- виртуальный портфель;
- сделки buy/sell;
- кэш прогнозов;
- обучение модели;
- сценарный макро-анализ;
- Swagger и документация на русском;
- Docker-конфигурация.

## Как кратко рассказывать проект

Можно рассказывать так:

1. Это учебная инвестиционная платформа с реальными рыночными данными, но без риска реальных денег.
2. Backend написан на FastAPI и управляет пользователями, портфелем, сделками и котировками.
3. Рыночные данные приходят из реальных открытых API: MOEX, Банк России и FRED.
4. Поверх этих данных работает ML-модуль многофакторной регрессии.
5. ML помогает оценивать, как изменение макрофакторов влияет на цену акции.
6. В итоге платформа не только показывает рынок, но и помогает пользователю учиться принимать инвестиционные решения.
